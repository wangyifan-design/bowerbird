# Bowerbird Landing Page

Landing page for [Bowerbird](https://bowerbird.ai) — an AI-powered real estate agent.

## Project Structure

```
bowerbird/
├── index.html              # Main landing page
├── assets/
│   └── images/
│       ├── hero-bg.jpg     # Hero section background
│       ├── card-01.png     # Highlight card 1 — No push. No bias. Just you.
│       ├── card-02.png     # Highlight card 2 — Knows your world
│       ├── card-03.png     # Highlight card 3 — Just talk. Anytime.
│       ├── card-04.png     # Highlight card 4 — Everywhere you look
│       └── card-05.png     # Highlight card 5 — Always learning
└── README.md
```

## Development

No build step required. Open `index.html` directly in a browser:

```bash
open index.html
```

Or serve locally with Python:

```bash
python3 -m http.server 8080
# then visit http://localhost:8080
```

## Deployment

Hosted via GitHub Pages. Push to `main` branch to deploy.
